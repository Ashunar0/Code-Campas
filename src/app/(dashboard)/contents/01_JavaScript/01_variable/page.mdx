## はじめに：変数って、プログラムの「箱」みたい！

プログラミングを学ぶ上で、避けて通れないのが<strong>「変数（へんすう）」</strong>です。変数と聞くと難しそうに感じるかもしれませんが、簡単に言えば「データをしまっておくための箱」のようなものです。

この「箱」にどんな種類のデータ（数字、文字など）をしまうのか、そしてその「箱」をどうやって用意するのか、この記事では JavaScript における変数の基本と、データの種類（データ型）について、初心者の方にも分かりやすく解説していきます。

## 変数を使ってみよう： let と const

JavaScript で変数を宣言するには、主に `let` と `const` という 2 つのキーワードを使います。それぞれの特徴を見ていきましょう。

### 1. let： 中身を「変更できる」箱

`let` は、後から中身を別のデータに変えたいときに使います。

```JavaScript

let userName = "山田"; // userName という箱に"山田"という名前を入れた
console.log(userName); // 出力: 山田


userName = "田中"; // userName の中身を"田中"に変更した
console.log(userName); // 出力: 田中

```

このように、let で宣言した変数は、プログラムの途中で値を<strong>再代入</strong>できます。

### 2. const：中身を「変更できない」箱（定数）

`const` は、<strong>「一度決めたら変えたくないデータ」</strong>を扱うときに使います。これは<strong>「定数（ていすう）」</strong>と呼ばれます。

```JavaScript

const age = 24; // age という箱に 24 を入れた
console.log(age); // 出力: 24

age = 25; // エラーになる！const で宣言した変数は変更できない

```

`const` で宣言した変数は、後から値を変更しようとするとエラーになります。変更されると困るような重要なデータや、アプリ内で常に同じ値を使うデータには `const` を使うのが一般的です。

### 補足：昔使われていた var について

JavaScript には `var` という変数宣言のキーワードもありますが、現在ではほとんど使われていません。`let` や `const` に比べて挙動が分かりにくい点があるため、今は `let` と `const` を使うのが主流です。

## データの種類：「データ型」を理解しよう！

変数には、さまざまな種類のデータを格納できます。これらのデータの種類を<strong>「データ型（データがた）」</strong>と呼びます。JavaScript にはたくさんのデータ型がありますが、まずはよく使う基本的なものから見ていきましょう。

### 1. 数値型（Number）

数字を扱うための型です。整数でも小数でも OK です。

```JavaScript

let age = 30; // 整数
let price = 100.5; // 小数

```

### 2. 文字列型（String）

文字や文章を扱うための型です。必ずダブルクォーテーション (") またはシングルクォーテーション (') で囲みます。

```JavaScript

let message = "こんにちは！";
let companyName = 'ABC 株式会社';

```

### 3. 真偽値型（Boolean）

「真（true）」か「偽（false）」のどちらかを表す型です。条件分岐などでよく使われます。

```JavaScript

let isStudent = true; // 学生である
let hasLicense = false; // 免許を持っていない

```

### 4. Null 型

「値がない」ことを明示的に示すための型です。

```JavaScript

let userData = null; // まだユーザーデータがない状態

```

### 5. Undefined 型

「値がまだ定義されていない」ことを示す型です。変数を宣言したけれど、まだ何も値が代入されていないときに自動的にこの型になります。

```JavaScript

let firstName; // 値を代入していないので undefined になる
console.log(firstName); // 出力: undefined

```

## まとめ

この記事では、JavaScript における変数の宣言方法（let と const）と、基本的なデータ型について解説しました。

- `let`: 後から値を変更したい変数に使う
- `const`: 値を固定したい定数に使う
- データ型: 変数に格納するデータの種類（数値、文字列、真偽値など）

変数は、プログラムを作る上でデータのやり取りに欠かせない、とても重要な概念です。まずは、実際にコードを書いてみて、いろいろなデータを変数に入れて試してみてくださいね！
